GitHub

GITHUB, es una plataforma de desarrollo colaborativo de software para alojar proyectos usando el sistema de control de versiones Git.

En primer lugar se debera crear el repositorio en GitHub, dando clic, en la opcion donde dice nuevo repositorio(Proyect Name) y de forma opcional aparece un campo de descripcion y un link a la web del proyecto. Tambien se puede indicar si el repositorio sera publico o no, por el cual solo ser aaccesible para el usuario que lo crea o los contribuidores.

Clonar Repositorio: se identifica por diversos procesos, pero esencialmente, para los casos en el que se presente un primer repositorio, por el cual se debe clonar uno existente. con esto se tiene una copia identica de un repositorio que se encuentre en un servidoe Remoto. por el cual al crearlo se puede generar por medio de una terminal donde seria mas eficiente desde el servidor para su accesibildiad.
